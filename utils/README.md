# Utilities

## 24-bit-color.sh

This script echoes a bunch of 24-bit color codes to the terminal to demonstrate
its functionality. Useful to test true color support for terminal.

```sh
bash 24-bit-color.sh
```

## moonlight-palette.html

Visual reference for the Moonlight color palette. Open in a browser to see all
color swatches with their hex codes organized by category (reds, oranges,
yellows, greens, teals, blues, purples, pinks, and grays).
